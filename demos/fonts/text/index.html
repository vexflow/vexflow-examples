Since the default @font-face rule uses font-display: swap, you should consider using a fallback font that is similar to your desired font (e.g., Georgia if you are trying to load Academico). The browser will briefly use the fallback font before your desired font is fully loaded. Or, you can use font-display: block in your own @font-face rule to avoid the font change.
